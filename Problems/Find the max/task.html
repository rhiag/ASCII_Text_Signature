Find the max Open on JetBrains Academy

Write a program that reads an array of ints and finds the index of the first occurrence of the maximum in the array.

The first line contains the number of elements in the array.
The second line contains the elements separated by spaces.
Guaranteed, that there is at least one element in the array.

Output a single integer value which is the index of the first maximum.


Sample Input:
5
2 5 3 4 5
Sample Output:
1

Sample Input:
6
1000 10 1000 10 10 1001
Sample Output:
5


Memory limit: 256 MB
Time limit: 8 seconds
